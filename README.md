# Beauty E-Commerce Platform

A comprehensive MERN stack (MongoDB, Express, React, Node.js) web application featuring secure JWT authentication, product browsing, shopping cart management, and order processing functionalities.

## Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JSON Web Tokens (JWT)

## Project Structure
- `/backend`: Node.js/Express server and REST API logic
- `/frontend`: React app with Tailwind CSS configuration

## Getting Started
Detailed setup instructions for both backend and frontend will be documented in their respective directories.
